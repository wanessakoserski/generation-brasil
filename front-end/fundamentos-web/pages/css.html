CSS

Definição

  CSS (Cascading Style Sheets) é usado para formatar o layout de uma página da web.

  Com CSS, você pode controlar a cor, a fonte, o tamanho do texto, o espaçamento entre os elementos, como os elementos são posicionados e dispostos, quais imagens ou cores de plano de fundo serão usadas, diferentes telas para diferentes dispositivos e tamanhos de telas e muito mais.

Como funciona?

O CSS pode ser adicionado aos documentos HTML de três maneiras:

  • Inline
    • Usando o atributo style dentro dos elementos HTML.
  • Interno
    • Usando um elemento <style> na <head> da página.
  • Externo
    • Usando um elemento <link> para vincular a um arquivo CSS externo.

Conceitos

Seletores

Os seletores podem ser: uma tag, um id de uma tag ou uma classe, ele serve para especificar no CSS qual elemento ou grupo de elementos será modificado através das propriedades CSS.

Propriedades

As propriedades de um código CSS, servem para dizer qual será o comportamento daquele seletor específico.

No exemplo anterior utilizei a propriedade font-size para determonar que todo parágrafo do meu HTML terá esse tamanho de fonte.
